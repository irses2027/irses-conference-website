# IRSES 2027 Conference Website Starter

This is the Phase 1 starter website for a professor's academic conference website.

## What is included

- A responsive conference homepage inspired by academic IEEE-style conference sites.
- Editable conference data in `src/data/conference.js`.
- DAU entrance image in `assets/daiict-entrance.png`.
- Sections for overview, important dates, tracks, committees, registration, venue, and contact.

## How to preview

Open `index.html` directly in a browser, or run a small local server from this folder:

```bash
python3 -m http.server 5173
```

Then visit:

```txt
http://localhost:5173
```

## Where to edit content

Most event content is controlled from:

```txt
src/data/conference.js
```

When the professor provides final details, update this file first.
